# Simple FIle organizer

Lightweight, multi-threaded file organizer built with Python to automatically organize messy directories. It sorts your files into logical subfolders (Documents, Images, Archives, etc.) based on their extensions.


- Automatically categorizes files using customizable mapping.
-Built with `tkinter`, featuring a clean, user-friendly interface.
-Uses multithreading to ensure the application stays responsive even while processing thousands of files.
-  Works seamlessly on Windows, macOS, and Linux thanks to `pathlib`.


1. Clone this repository to your machine.
2. Run the script: `python organizer.py`.
3. Click "Select Folder & Organize".
4. Watch the progress bar as your files are moved into their new homes!


- *Python 3.x*
- *Tkinter*: For the GUI implementation.
- *Pathlib*: For robust, object-oriented file system paths.
- *Threading*: To handle background operations without freezing the UI.


Built it to master some of the python concepts and modules. :)
